# Dashboard 重新设计 - 静谧简约风格

## 🎨 设计理念

**"Serene Minimalism"（静谧简约）**

这是一个极度克制、优雅精致的设计，体现"少即是多"的极简主义哲学。每个元素都有其存在的理由，通过大量留白、精确的排版和微妙的动画，创造出一个宁静而专业的工作环境。

## 🎯 设计特点

### 1. 字体系统

**主字体：IBM Plex Sans**
- 现代、清晰、优雅
- 极细的标题字重（300）创造轻盈感
- 适中的正文字重（400-500）保证可读性

**等宽字体：JetBrains Mono**
- 用于数字和代码展示
- 科技感强，与项目管理主题契合
- 清晰的数字辨识度

### 2. 色彩方案

**中性灰色系**
```css
背景色：#fafafa（极浅灰）
表面色：#ffffff（纯白）
主文本：#1a1a1a（深灰黑）
次文本：#6b7280（中性灰）
三级文本：#9ca3af（浅灰）
强调色：#64748b（蓝灰色）
```

**设计原则**
- 避免使用鲜艳颜色
- 通过灰度层次建立视觉层级
- 仅在必要时使用柔和的强调色

### 3. 布局与留白

**极简布局**
- 最大宽度：1400px
- 大量纵向留白（72px）
- 卡片间距：24px
- 内容区域宽松的内边距

**不对称网格**
```
统计卡片：2:1:1:1 的比例
- 项目总数：占据双倍宽度，强调重要性
- 其他指标：等宽排列
```

### 4. 视觉细节

**极细边框**
- 1px 的浅灰色边框（#f3f4f6）
- 悬停时变深（#e5e7eb）
- 顶部装饰线：2px 渐变（悬停时显示）

**柔和阴影**
```css
静态：0 1px 3px rgba(0, 0, 0, 0.04)
悬停：0 4px 12px rgba(0, 0, 0, 0.06)
```

**圆角**
- 卡片：12px（柔和但不过度）
- 小元素：6-8px

### 5. 微交互动画

**页面加载**
```
整体：淡入 + 向上滑动（0.8s）
标题：向下滑动（0.6s，延迟 0.1s）
统计区：向上滑动（0.6s，延迟 0.2s）
类型区：向上滑动（0.6s，延迟 0.3s）
操作区：向上滑动（0.6s，延迟 0.4s）
底部：淡入（0.6s，延迟 0.5s）
```

**悬停效果**
- 轻微上移（-2px）
- 阴影加深
- 边框颜色变化
- 图标缩放（1.05倍）
- 箭头淡入滑动

**指示器动画**
- 脉动效果（2s 循环）
- 不透明度和缩放变化
- 传达"实时"感

## 🎭 交互细节

### 统计卡片

**悬停状态**
1. 卡片轻微上浮（-2px）
2. 阴影加深
3. 边框颜色变深
4. 顶部装饰线淡入（渐变效果）

**项目总数卡片**
- 特大号数字（72px）
- 实时指示器（脉动蓝点）
- 占据更大空间，强调重要性

### 类型卡片

**组成元素**
1. 图标区域：40x40px，柔和背景
2. 类型名称 + 数值（等宽字体）
3. 进度条：4px 高度，柔和渐变填充

**悬停效果**
- 图标向上浮动（-2px）
- 图标颜色加深
- 整体卡片阴影增强

### 快速操作卡片

**初始状态**
- 图标：灰色，柔和背景
- 箭头：隐藏（opacity: 0）

**悬停状态**
1. 卡片上浮
2. 图标背景变为强调色
3. 图标颜色变白
4. 图标轻微放大
5. 箭头淡入并滑动出现
6. 背景渐变覆盖层淡入

## 📱 响应式设计

### 桌面端（> 1200px）
```
统计网格：2:1:1:1
类型网格：1:1
操作网格：1:1:1
```

### 平板端（768-1200px）
```
统计网格：2x2
类型网格：1:1
操作网格：1:1（第三项独占一行）
```

### 移动端（< 768px）
```
统计网格：1列
类型网格：1列
操作网格：1列
标题字号：48px → 36px
统计数字：72px → 56px
内边距：72px → 48px
```

## 🌙 深色模式支持

**自动适配系统主题**
```css
@media (prefers-color-scheme: dark)
```

**深色调色板**
- 背景：#0f0f0f（极深灰）
- 表面：#1a1a1a（深灰）
- 文本：#f5f5f5（浅灰白）
- 次文本：#a3a3a3
- 边框：#2a2a2a

## ♿ 可访问性

**动效敏感性**
```css
@media (prefers-reduced-motion: reduce)
```
对于设置了"减少动画"的用户，所有动画将降至最低

**键盘导航**
- 所有交互元素可通过 Tab 键访问
- 清晰的焦点指示器

**语义化 HTML**
- 正确的标题层级（h1, h2）
- 语义化的 section 和 header 标签
- 有意义的 button 元素

## 🎨 设计哲学

**少即是多**
- 移除所有不必要的装饰
- 每个元素都服务于功能
- 通过留白创造呼吸感

**优雅的层次**
- 通过字号、字重、颜色建立层次
- 避免使用过多视觉层级
- 让用户专注于重要信息

**微妙的惊喜**
- 悬停时的微动画
- 渐进式的加载动画
- 不打扰但有存在感

**专业而温暖**
- 中性色调传达专业感
- 柔和的圆角和阴影增加亲和力
- 清晰的排版保证可读性

## 🚀 技术实现

**CSS 架构**
- CSS 变量统一管理样式
- 模块化的 CSS 类命名
- 移动优先的响应式设计

**性能优化**
- 使用 CSS 动画（GPU 加速）
- 避免复杂的 JavaScript 动画
- 按需加载字体

**浏览器兼容**
- 支持所有现代浏览器
- 优雅降级（不支持的浏览器显示静态版本）
- 打印样式优化
